🧮 Calculadora em Python
Calculadora simples desenvolvida em Python para praticar lógica de programação e estruturas condicionais.
💡 Funcionalidades

Adição
Subtração
Multiplicação
Divisão (com tratamento de divisão por zero)

🚀 Como executar
Pré-requisitos: Python 3 instalado na máquina.
bash# Clone o repositório
git clone https://github.com/Daniel5558/Calculadora---Python

# Acesse a pasta
cd Calculadora---Python

# Execute o programa
python calculadora.py
🖥️ Exemplo de uso
== CALCULADORA ==
Digite o primeiro número: 10
Digite o segundo número: 2
Escolha a operação:
1 - Adição
2 - Subtração
3 - Multiplicação
4 - Divisão
Digite o número da operação desejada: 4
Resultado: 5.0
